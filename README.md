### 项目构建
> 1. 拉取代码

``` shell
git clone https://github.com/maliaoMJ/hexo-blog.git ./hexo-blog

```

> 2. 安装依赖
```shell
cd hexo-blog
yarn or npm install
hexo server
```
> 3. 查看效果
打开你的游览器 http://localhost:4000
